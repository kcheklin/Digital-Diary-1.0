# 📔 E-Diary System
> A JavaFX-based personal diary with smart backup and auto-cleanup logic.

---

### ⚡ Quick Features
| Feature | Description |
| :--- | :--- |
| **🎨 CRUD** | Create, View, Edit, and Search diary entries. |
| **🎭 Mood Tracker** | Visual emotional tracking via sliders. |
| **🖼️ Media** | Local image attachments with auto-rename logic. |
| **♻️ Smart Bin** | 30-day auto-cleanup & one-click restoration. |

---

### 🛠️ Tech Stack
* **Language:** Java 17+ (JavaFX)
* **Storage:** Local CSV (Custom Escaping Engine)
* **Concurrency:** `ScheduledExecutorService` for background tasks.

---

### 📂 File Structure
Data is stored at: `D:/users/{username}/MyDiaryApp/`
> **Note:** If you are on Mac/Linux, please update the file path in controllers.

---

### 🚀 How to Run
1. Ensure **JDK 17** and **JavaFX SDK** are configured.
2. Clone this repo.
3. Run `Main.java`.
